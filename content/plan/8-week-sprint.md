## 每日節奏（90 分鐘）

```
09:00 - 09:25  讀一個 section（NSCA 書 by_book/）              [25 min]
09:25 - 09:30  散步 / 喝水 / 切換到發散模式                     [5 min]
09:30 - 09:55  free recall（閉眼講給自己聽 / 用 Talos 出選擇題）[25 min]
09:55 - 10:00  發散模式                                         [5 min]
10:00 - 10:25  寫筆記（content/notes/ 或 content/mastered/）    [25 min]
10:25 - 10:30  收尾 + 更新 data/topics.yaml 狀態                [5 min]
```

---

## 每週節奏

- **週一**：啟動該週第一個 topic（status: todo → learning）
- **週二 ~ 週六**：每天一個新 topic + 前一天的複習
- **週日**：當週全部 topic 做一輪 cross-chapter application（換情境套用）

---

## 間隔重複節奏（每個 topic）

| 次 | 何時 | 動作 |
|----|------|------|
| T+0 | 讀完當天 | 寫筆記 + status: learning |
| T+1 | 隔天 | free recall（5 題） |
| T+3 | 第 3 天 | practice test（Talos 出 10 題選擇） |
| T+7 | 第 7 天 | 變式練習（套用到其他族群 / 章節） |
| T+14 | 第 14 天 | teach back（解釋給 Talos 聽） |
| T+30 | 第 30 天 | 跨章節整合（考前總複習） |

`data/topics.yaml` 的 `next_review` 跟 `review_count` 自動推進。

## 模擬考規劃

### 第一次（W7 週一，08-06）

- 官方 NSCA Practice Exam（150 題，付費）
- 或：自己組 50 題選擇題（從 by_book 各章節抽）
- **時間限制**：依官方 200 題 / 4 小時比例
- **做完不對答案**：先預測自己對幾題

### 第二次（W7 週四，08-09）

- 換一套題庫（避免背答案）
- 對照第一次找弱點

### 第三次（W7 週日，08-12）

- 限時全真模擬

### 錯題整理

每題錯的 → 寫到 `content/notes/YYYY-MM-DD-mistake-chXX.md`：

```
題目：[題幹]
我的答案：
正確答案：
為什麼錯：
（這個 topic 我卡在哪：）
```

## 考前衝刺（W8）

- **不讀新東西**
- **只做三件事**：錯題本 / 公式口訣 / Talos 模擬面試
- **考前一天（08-19 週三）**：完全休息，散步、看電影、睡飽

---

## 風險與退路

| 風險 | 退路 |
|------|------|
| W5 進度落後 | 砍掉 ch19 (speed/agility) 跟 ch22 (rehab) 詳細，剩概念性了解 |
| 模擬考 < 60% | 把 W8 改為再讀一輪弱項，模擬考延後 |
| 工作太忙 | 每天 90 min 縮為 30 min，但**不中斷節奏** |
| 健康出問題 | 暫停所有複習，先養身體，調整考試日期 |
